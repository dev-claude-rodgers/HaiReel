RouteJin プライバシーポリシー

本プライバシーポリシーは、RODGERS（以下「当方」）が提供するAndroidアプリケーション「RouteJin」（以下「本アプリ」）における個人情報の取り扱いについて説明します。

1. 収集する情報

本アプリが収集・利用する情報は以下のとおりです。

  • 位置情報：配達先の地図表示・現在地表示のために、デバイスのGPS情報を利用します。位置情報は外部サーバーには送信されません。
  • カメラ：伝票スキャン機能のために、カメラを使用します。撮影した画像は端末内でのみ処理され、外部には送信されません。
  • 入力データ：運転者名・配達先住所・日報データ・点呼記録など、ユーザーが入力した業務情報を端末内に保存します。

2. 情報の利用目的

  • 配達先住所のジオコーディング（Google Geocoding API を使用）
  • 地図上への配達先表示（Google Maps SDK を使用）
  • 業務報告書・点呼記録簿のExcelファイル出力
  • アプリの不具合検出・品質改善（Firebase Crashlytics を使用）

3. 第三者への情報提供

当方は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく場合

本アプリは以下の外部サービスを利用しています。各サービスのプライバシーポリシーもご確認ください。

  • Google Maps Platform（Google LLC）
  • Firebase Crashlytics（Google LLC）

4. データの保存場所

本アプリが収集・生成するデータ（日報・点呼記録・配達先情報）は、すべてユーザーのデバイス内に保存されます。当方のサーバーには送信・保存されません。

バックアップパスワードは、Androidの暗号化SharedPreferences（AES256-GCM）を使用して端末内に暗号化保存されます。

5. 権限について

  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION：現在地の地図表示に使用します。
  • CAMERA：伝票スキャン機能に使用します。
  • WRITE_EXTERNAL_STORAGE / READ_EXTERNAL_STORAGE：Excelファイルのダウンロードフォルダへのエクスポートおよびバックアップに使用します。
  • INTERNET：Google Maps・ジオコーディング・Crashlyticsへのアクセスに使用します。
  • RECEIVE_BOOT_COMPLETED / POST_NOTIFICATIONS：点呼リマインダー通知に使用します。

6. 未成年者のプライバシー

本アプリは業務用途を想定しており、18歳未満の方を対象としていません。

7. プライバシーポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後は本ページに掲載します。
